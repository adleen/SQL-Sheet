# **App Name**: SQL Sheet

## Core Features:

- Excel Upload: Allow users to upload .xlsx files.
- Table Preview: Display a preview of the uploaded Excel table.
- SQL Conversion: Generates SQL INSERT statements from the Excel data.
- SQL Download: Provide options for users to copy SQL statements to the clipboard, or download SQL files containing the insert statements.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional, reliable feel.
- Background color: Light gray (#F5F5F5), offering a clean, neutral backdrop.
- Accent color: Teal (#009688) to draw attention to important CTAs.
- Body and headline font: 'Inter' (sans-serif) to maintain a clean and easily-readable design.
- Use simple, professional icons for actions like uploading, converting, and downloading.
- Prioritize a clear layout that focuses on the table preview and SQL output.
